`wget https://github.com/LasVegasCoder/FireWALL/blob/master/princeserverguard.ipt`

`chmod +x princeserverguard.ipt`

`./princeserverguard.ipt`

Then watch your traffic with:
`watch -d -n1 iptables -vnL`
